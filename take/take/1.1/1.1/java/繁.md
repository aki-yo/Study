## docker


```
# 强制删除所有容器（包括运行中的容器）
docker rm -f $(docker ps -aq)

# 分步解释：
# - `docker ps -aq`：列出所有容器的 ID（-a 包含停止的容器，-q 只返回 ID）
# - `docker rm -f`：强制删除容器（-f 可删除运行中的容器）
```



如果一个网路是外部已经创建好的，那compose中需要
```
networks:
  ohh:
    external: true  # 声明使用外部网络
```
 或者
```
networks:
  be-ohh:
    name: ohh    # 直接指定外部网络名称
    external: true  # 声明使用外部网络
```
